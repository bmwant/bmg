Welcome to the BMG!

BMG - is a code generator for your Python web-projects. It uses Bottle framework, Peewee object-relations mapper, Jinja2 templating, WTForms for your forms validation and CSS-framework of your choice as a foundation to create basic project structure (directories tree and blueprint code files) and to generate boilerplate code for CRUD operations with your models.

If you don't have project yet you can create your own in a few steps with Create button. Also you can generate forms, views and controllers for your existing models-file or even generate it based on your database tables with Generate button. Let's go.
